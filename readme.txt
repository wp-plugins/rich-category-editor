== Installation ==

Download, Upgrading, Installation:

Upgrade

1. First deactivate Rich Category Editor
1. Remove the `rich-category-editor` directory

**Install**

1. Unzip the `rich-category-editor.zip` file. 
1. Upload the the `rich-category-editor` folder (not just the files in it!) to your `wp-contents/plugins` folder. If you're using FTP, use 'binary' mode.

**Activate**

1. In your WordPress administration, go to the Plugins page
1. Activate the Rich Category Editor plugin and you will see TiniMCE on your category/tag edit page.

If you find any bugs or have any ideas, please mail me.
